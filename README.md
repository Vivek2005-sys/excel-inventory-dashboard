# Excel Inventory Dashboard

## Project Overview
This project is an Excel-based inventory management dashboard built using a dataset of 500 records.  
The dashboard helps analyze product stock levels and supplier inventory value.

## Features
- Inventory summary metrics
- Product stock distribution chart
- Supplier inventory value chart
- Interactive slicers for filtering

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel formulas



## Dataset Size
500 rows of inventory data.
